预检 ​

您可以从配置中注入原始 css 作为预处理。解析的 theme 可用于自定义 css。

ts
preflights: [
  {
    getCSS: ({ theme }) => `
      * {
        color: ${theme.colors.gray?.[700] ?? '#333'};
        padding: 0;
        margin: 0;
      }
    `
  }
]
